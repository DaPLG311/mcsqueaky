# McSqueeky™

Premium recurring residential cleaning for the 518 / Capital Region, NY.
Marketing site — static HTML/CSS/JS, **no build step**.

🔗 Production: https://mcsqueeky.vercel.app

## Stack
- Plain HTML + `styles.css` + `script.js`
- Fonts: Fraunces (display) + Inter (body) via Google Fonts
- SEO: per-page meta + canonical, Open Graph, `LocalBusiness`/`Service` + `FAQ` JSON-LD, `robots.txt`, `sitemap.xml`, `site.webmanifest`

## Pages
`index.html` · `recurring-cleaning.html` · `deep-cleaning.html` · `move-in-move-out.html` · `services.html` · `service-areas.html` · `cleaning-albany-ny.html` (city template) · `about.html` · `faq.html` · `quote.html` · `privacy.html` · `terms.html` · `404.html`

## Deploy
Push to `main` → Vercel auto-deploys to production. No CLI needed.

```sh
git add -A && git commit -m "Update site" && git push origin main
```
